# CSE 13S Assignment 4     

## Build    
    
    $ make     
    
    or
    
    $ make all
    
## Running    
    $ ./tsp -[hdi:o:]    
    
    
## COMMAND LINE OPTIONS:    
* -i: Sets the file to read form (input file).    
- -o: Sets the file to write to (output file). 
* -h: Prints a help message to stdio.
- -d: Treats all graphs as directed.
   



## Cleaning    
    
    $ make clean
